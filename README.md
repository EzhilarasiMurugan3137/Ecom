# Ecom Automation Framework 🚀
Demo Web Shops complete Automation Framework
### Project Overview
This is a professional-grade Selenium automation framework designed for the **Demo Web Shop** e-commerce site. It demonstrates the implementation of a scalable **Page Object Model (POM)** architecture to handle end-to-end testing of retail workflows.

### 🛠 Tech Stack
* **Language:** Java
* **Automation:** Selenium WebDriver
* **Framework:** TestNG
* **Design Pattern:** Page Object Model (POM)
* **Reporting:** Extent Reports 5.x
* **Build Tool:** Maven
* **CI/CD:** Jenkins Integrated

### ✨ Key Features
* **Listener Integration:** Automated screenshot capture on test failures via TestNG Listeners.
* **Hybrid Framework:** Modular design with a dedicated `BaseClass` for driver initialization.
* **Dynamic Wait Management:** Robust handling of AJAX elements and page loads.
* **Clean Reporting:** Interactive HTML reports generated after every suite execution.

### 📊 Test Suite Coverage
The **Ecom Regression Suite** includes:
1. User Registration & Validation
2. Login (Positive/Negative)
3. Product Search Functionality
4. Category Navigation
5. Add to Cart & Quantity Verification
6. Wishlist Management
7. Cart-to-Order Workflow (Checkout)

### 🚀 How to Run
1. Clone the repository: `git clone https://github.com/EzhilarasiMurugan3137/Ecom.git`
2. Open in Eclipse/IntelliJ.
3. Right-click `testng.xml` -> **Run as TestNG Suite**.
